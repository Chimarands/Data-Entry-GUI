# Data-Entry-GUI
Python Tkinter Data Entry to CSV
A simple desktop application built with Python and Tkinter that allows users to enter personal data through a GUI form and save it to a CSV file.

Features
Clean and simple graphical user interface (GUI)
Input fields for Name, Age, and Email
Saves entries to a CSV file with a single button click
Automatically clears input fields after saving
Appends new entries without overwriting existing data
Requirements
Python 3.x
Tkinter (included with standard Python installation)
